# 领域术语表

> 统一业务与技术的沟通用语，避免 AI 与团队对同一概念理解不一致。
> 产品背景见 [`product-overview.md`](product-overview.md)。代码 / 契约映射待 F2 落地后补充。

| 术语 | 定义 |
|---|---|
| 虚拟人 Persona | 有固定人设（性格 / 背景 / 说话风格）的虚构角色，绑定一座设定城市与一个家坐标 |
| 设定城市 | 运营 / 系统为虚拟人指定的一座真实城市，ta 的一天都在这城市真实地图上展开 |
| 家 Home | 虚拟人在设定城市的真实坐标点，作息锚点 |
| 设定地点 Place | 虚拟人活动的真实地点（家 / 常去的店…），各带真实经纬度 |
| 事件 Event | 时间线原子节点：某虚拟人在某现实时间、某地点（或在途）、做某事；不可变 |
| 移动 / 在途事件 | 特殊事件：从 A 前往 B，按真实路网估时；地图上表现为移动中 |
| 时间线 Timeline | 某虚拟人某天的有序事件序列（派生视图） |
| 现实时间锚点 | 事件对应真实时刻，按设定城市时区推进 |
| 真实情境 | 生成时叠加的真实信号：天气、POI 营业时间、工作日 / 周末 / 节假日 |
| 事件自链 | 每个事件带结束时间，到点自动触发生成下一件，永续推进 |
| AI Gateway（自建） | generation-service 与 LLM 之间的自建网关：集中 LLM 密钥、限流 / 重试 / fallback、缓存、成本观测、可选 guardrails |
| 地图锚点 | 事件地点经纬度，用于地图展示与轨迹连线 |
| 状态 | 连贯性判定标签（在家 / 外出（在途）/ 用餐 / 休闲 / 睡眠…） |
