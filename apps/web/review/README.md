# review/ — AI 自检 review 报告

> AI Agent 在 PR 前生成的自检报告，基于 spec / plan / diff / rules。
> **仅供评审和提测前参考，不替代人工评审或测试验收。**

## 命名

`YYYY-MM-DD-review-<标识>.md`，与对应任务同标识。

## 必填字段

- **对照结论**：对应 spec / plan 链接。
- **已改文件**清单。
- **风险点**：越界 / 高影响 / 禁止项 / 无依据，分类列出。
- **是否需要人工确认**：是否触及高影响区 / 命中 review-gates。
- **下一步建议**。
