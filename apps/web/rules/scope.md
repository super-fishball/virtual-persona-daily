# 可修改范围与边界 — apps/web

## 可改

- `apps/web/` 内的源码、样式、前端测试、本子项目文档。

## 不可改（跨边界）

- 任何 `servers/*`、其他 `apps/*`、根 `rules/`、根 `context/`。
- 跨子项目契约（调用的后端接口）——只能**按 feature 层既定契约对接**，需求变更须回到 feature 层。

## 越界处理

发现任务需要改动 `apps/web` 之外的内容 → **暂停**，提示这是跨子项目/跨层改动，需走 feature 层或转交对应负责人。
