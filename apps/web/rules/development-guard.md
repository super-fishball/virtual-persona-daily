# 开发中自检 — apps/web（栈特有补充）

> 继承根 [`../../../rules/development-guard.md`](../../../rules/development-guard.md) 的通用自检流程（对照物 / 六类清单 / 命中即停 / 输出格式）。
> 本文件**只列 apps/web（前端栈）特有的额外检查步骤**。

## 前端特有检查

- 是否绕过统一请求封装，直接发起跨子项目调用？
- bundle / 前端代码是否混入密钥、token？
- 是否用"前端隐藏"冒充权限控制（本应后端把关）？
- 路由守卫 / 登录态处理是否被破坏？
