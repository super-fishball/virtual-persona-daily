# 额外禁止项 — servers/api

> 继承根 [`../../../rules/forbidden.md`](../../../rules/forbidden.md) 全部条款，以下为 servers/api 额外约束（只增不减）。

- 禁止单方变更**已被前端依赖的既有契约**（须经 feature 层集体定稿）。
- 禁止在无迁移脚本/无回滚方案的情况下改动数据库 schema。
- 禁止跳过认证授权校验、或在日志中输出敏感数据（凭证、完整身份信息）。
- 禁止直接操作生产数据库（呼应根 forbidden 的生产操作禁令）。
