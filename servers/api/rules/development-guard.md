# 开发中自检 — servers/api（栈特有补充）

> 继承根 [`../../../rules/development-guard.md`](../../../rules/development-guard.md) 的通用自检流程（对照物 / 六类清单 / 命中即停 / 输出格式）。
> 本文件**只列 servers/api（后端栈）特有的额外检查步骤**。

## 后端特有检查

- 数据库 schema 变更是否带迁移与回滚方案？
- 新增 / 变更接口是否有鉴权与输入校验？
- 是否单方改动了已被前端依赖的既有契约？
- 事务边界 / 并发控制是否正确？
- 日志是否泄露敏感数据？
